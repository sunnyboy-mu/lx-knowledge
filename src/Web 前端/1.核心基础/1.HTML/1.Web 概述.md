## 专业术语

- **Web**: 互联网
- **W3C**: 万维网联盟，非盈利的组织；为互联网提供各种标准
- **XML**: 可拓展标记语言`Extension Markup Language`，用于定义文档结构
- **MDN**: Mozilla Development Network，Mozilla 开发者社区

## 什么是 HTML ?

HTML（`Hyper Text Markup Language` 超文本标记语言） 是 W3C 组织定义的语言标准；用于描述页面结构的语言

通俗来说就是有什么东西，该东西表示什么含义

> **XHTML**: 可以认为是 HTML 的一种版本，完全符合 XML 的规范

## 什么是 CSS ?

CSS（`Cascading Style Sheets` 层叠样式表）是 W3C 组织定义的语言标准；用于描述页面展示的语言。

通俗来说就是决定页面长什么样子

## 浏览器

浏览器分为：

- Shell 外壳
- Core 内核（JS 执行引擎、渲染引擎）

**主流浏览器及其内核**

- IE: Trident
- Firfox: Gecko
- Chrome: Blink
- Safari: Webkit
- Opera: Presto / Blink

注：大家可能会疑惑，为什么没有国产浏览器呢？在这里我们所说的主流，一定要有自己内核的浏览器

> 小知识：国产浏览器大都基于`Blink`内核，加上自定义的`Shell`外壳开发而来

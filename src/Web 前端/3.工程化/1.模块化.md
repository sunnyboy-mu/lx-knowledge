## 01、模块化标准

- CommonJS（CJS）
- AMD
- CMD
- UMD
- Ecmascript Module（ESM）

---

CJS、AMD、CMD、UND 是：民间标准；ESM 是：官方标准。

广泛使用：CJS、ESM

## 02、CJS 与 ESM 核心区别

- 来源：CJS 民间标准；ESM 官方标准
- 方式：CJS 运行时；ESM 编译时

---

- 运行时：运行后才能确定依赖关系

```js
if (xxx) {
  const xxx = require("./2.js");
} else {
  const xxxx = require("./3.js");
}
```

- 编译时：运行之前确定依赖关系

```js
// 语法错误
if(xxx){
    import xxx form './2.js';
}else{
    import xxxx form './3.js';
}

// 正确
import xxx form './2.js';
import xxxx form './3.js';
```
